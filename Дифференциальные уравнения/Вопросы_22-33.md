# Вопрос 22. Особые точки. Узел

Для системы x'=Ax.
Собственные значения вещественные одного знака.
Если отрицательны — устойчивый узел, если положительны — неустойчивый.

# Вопрос 23. Седло

Собственные значения разных знаков.
Особая точка всегда неустойчива.

Доказательство: существуют траектории, удаляющиеся от равновесия вдоль собственного направления положительного собственного значения.

# Вопрос 24. Вырожденный и дикритический узел

Кратный корень характеристического уравнения.
Возможны один или два линейно независимых собственных вектора.

# Вопрос 25. Центр

Комплексные корни:
λ=±iβ.

Траектории — замкнутые кривые.
Устойчив, но не асимптотически устойчив.

# Вопрос 26. Фокус

λ=α±iβ.

Если α<0 — устойчивый фокус.
Если α>0 — неустойчивый фокус.

# Вопрос 27. Вырожденные случаи

Определитель матрицы равен нулю либо имеются нулевые собственные значения.
Линейное приближение недостаточно.

# Вопрос 28. Особые точки нелинейных автономных систем

Находятся из условий:

f(x,y)=0,
g(x,y)=0.

Исследование выполняется по матрице Якоби.

# Вопрос 29. Устойчивость полиномов. Необходимое условие

Для устойчивости все коэффициенты характеристического полинома должны быть одного знака.

# Вопрос 30. Критерий Рауса–Гурвица

Все корни полинома имеют отрицательные действительные части тогда и только тогда, когда все главные определители матрицы Гурвица положительны.

# Вопрос 31. Первые интегралы автономных систем

Функция V(x), сохраняющаяся вдоль траекторий:

∇V·f(x)=0.

Тогда

V(x)=C.

# Вопрос 32. Линейные уравнения в частных производных первого порядка

Общий вид:

A(x,y)u_x+B(x,y)u_y=C(x,y,u).

Основной метод решения — метод характеристик.

# Вопрос 33. Квазилинейные уравнения первого порядка

Общий вид:

A(x,y,u)u_x+B(x,y,u)u_y=C(x,y,u).

Решаются методом характеристик:

dx/A=dy/B=du/C.
