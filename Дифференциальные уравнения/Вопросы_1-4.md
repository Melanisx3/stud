# Вопрос 1. Системы ОДУ. Задача Коши

## Определение
Нормальная система:
\[
\dot x=f(t,x),\quad x\in R^n.
\]

Задача Коши:
\[
x(t_0)=x_0.
\]

## Теорема существования и единственности
Если функция f непрерывна и удовлетворяет условию Липшица по x в некоторой области, то существует единственное решение задачи Коши.

## Геометрический смысл
Через каждую точку проходит единственная интегральная кривая.

# Вопрос 2. Линейные однородные системы

\[
\dot x=A(t)x.
\]

Свойства:
1. Сумма решений является решением.
2. Произведение решения на константу является решением.
3. Линейная комбинация решений является решением.
4. Множество решений образует линейное пространство размерности n.

# Вопрос 3. Фундаментальная матрица

Матрица Φ(t), составленная из линейно независимых решений системы.

Критерий фундаментальности:

\[
\det \Phi(t)\neq0.
\]

Тогда столбцы образуют фундаментальную систему решений.

# Вопрос 4. Свойства фундаментальных матриц

1. Определитель не обращается в нуль.
2. Любые две фундаментальные матрицы связаны соотношением

\[
\Phi_2(t)=\Phi_1(t)C.
\]

3. Общее решение:

\[
x=\Phi(t)C.
\]

4. Для матрицы Коши:

\[
X(t,t_0)=\Phi(t)\Phi^{-1}(t_0).
\]
