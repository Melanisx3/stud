# 4. Свойства фундаментальных матриц

Пусть Φ(t) — фундаментальная матрица системы

x'=A(t)x.

## Свойства

1. detΦ(t)≠0.

2. Любые две фундаментальные матрицы связаны соотношением

Φ₂(t)=Φ₁(t)C,

где C — постоянная невырожденная матрица.

3. Общее решение имеет вид

x(t)=Φ(t)C.

4. Существует обратная матрица Φ⁻¹(t).

5. Матрица Коши:

X(t,t₀)=Φ(t)Φ⁻¹(t₀).

## Значение матрицы Коши
Позволяет записать решение задачи Коши:

x(t)=X(t,t₀)x₀.
