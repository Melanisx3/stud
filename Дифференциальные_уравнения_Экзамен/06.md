# 6. Линейные системы с постоянными коэффициентами

## Определение
Линейной системой с постоянными коэффициентами называется система

x' = Ax,

где A — постоянная матрица порядка n×n.

## Общий метод решения
Решение ищется в виде

x=v e^{λt},

где v — постоянный вектор.

Подставляя в систему, получаем

(A-λE)v=0.

Для существования ненулевого решения необходимо

|A-λE|=0.

Это характеристическое уравнение системы.

## Алгоритм решения
1. Составить характеристическое уравнение.
2. Найти собственные значения λ.
3. Для каждого λ найти собственный вектор.
4. Построить фундаментальную систему решений.
5. Записать общее решение.

## Пример
x'=2x.

Характеристическое уравнение:

λ-2=0.

λ=2.

Общее решение:

x=Ce^{2t}.

## Теорема
Если матрица A имеет n линейно независимых собственных векторов, то общее решение имеет вид линейной комбинации решений

x_k=v_k e^{λ_k t}.
