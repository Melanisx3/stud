# 7. Простые собственные значения. Построение общего решения

## Теорема
Пусть характеристическое уравнение имеет n различных корней

λ₁, λ₂, ..., λₙ.

Тогда каждому собственному значению соответствует собственный вектор vₖ.

Решения имеют вид

xₖ=vₖe^{λₖt}.

Эти решения линейно независимы и образуют фундаментальную систему решений.

## Общее решение

x=C₁v₁e^{λ₁t}+...+Cₙvₙe^{λₙt}.

## Доказательство
Линейная независимость следует из линейной независимости собственных векторов, соответствующих различным собственным значениям.

## Пример
Система

x'=3x,
y'=y.

Корни:

λ₁=3,
λ₂=1.

Общее решение:

x=C₁e^{3t},
y=C₂e^{t}.
