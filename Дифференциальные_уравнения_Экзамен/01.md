# 1. Системы обыкновенных дифференциальных уравнений. Задача Коши

## Определение системы ОДУ
Система первого порядка называется нормальной системой:

x' = f(t,x),

где x=(x1,...,xn).

## Задача Коши
Найти решение системы, удовлетворяющее начальному условию

x(t0)=x0.

## Теорема Пикара–Линделёфа
Если функция f(t,x):

1. непрерывна;
2. удовлетворяет условию Липшица по x,

то существует единственное решение задачи Коши в некоторой окрестности точки (t0,x0).

## Геометрический смысл
Через каждую точку области проходит единственная интегральная кривая.

## Условие Липшица
||f(t,x)-f(t,y)||≤L||x-y||.

## Следствие
При малом изменении начальных данных решение изменяется непрерывно.
