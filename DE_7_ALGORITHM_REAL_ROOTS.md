# 7️⃣ Алгоритм построения ФСР для линейной однородной системы n-го порядка (вещественные корни)

## 📖 Определение

Пусть $\lambda$ — **вещественное собственное значение** матрицы $A$ кратности $m$ (может быть кратность > 1).

---

## ✅ Случай 1: Простое собственное значение (кратность = 1)

Если $\lambda$ — простое собственное значение, то каждому соответствует решение:

$$\mathbf{x}(t) = e^{\lambda t} \mathbf{v}$$

где $\mathbf{v}$ — собственный вектор: $(A - \lambda I)\mathbf{v} = \mathbf{0}$.

---

## ✅ Случай 2: Кратное собственное значение

Если $\lambda$ — собственное значение кратности $m$, то ему могут соответствовать несколько решений:

### **Подслучай 2a: Матрица диагонализируема**

От $\lambda$ исс $m$ линейно независимых собственных векторов $\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_m$.

Решения:
$$\mathbf{x}_i(t) = e^{\lambda t} \mathbf{v}_i, \quad i = 1, 2, \ldots, m$$

### **Подслучай 2b: Матрица недиагонализируема**

От $\lambda$ исс меньше $m$ линейно независимых собственных векторов.

Нужно использовать **обобщенные собственные векторы** (векторы Нейманна).

---

## 📐 Обобщенные собственные векторы

**Определение:** Вектор $\mathbf{w}$ называется **обобщенным собственным вектором** (ранга $k$) для $\lambda$, если:

$$(A - \lambda I)^k \mathbf{w} = \mathbf{0}$$

но $(A - \lambda I)^{k-1} \mathbf{w} \neq \mathbf{0}$.

---

## 📝 Примеры

### Пример 1: Три разных вещественных корня

$$\frac{d\mathbf{x}}{dt} = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 2 & 0 \\ 0 & 0 & 3 \end{pmatrix} \mathbf{x}$$

Собственные значения: $\lambda_1 = 1, \lambda_2 = 2, \lambda_3 = 3$

Собственные векторы: 
$$\mathbf{v}_1 = \begin{pmatrix} 1 \\ 0 \\ 0 \end{pmatrix}, \mathbf{v}_2 = \begin{pmatrix} 0 \\ 1 \\ 0 \end{pmatrix}, \mathbf{v}_3 = \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix}$$

ФСР:
$$\mathbf{x}_1(t) = e^t \begin{pmatrix} 1 \\ 0 \\ 0 \end{pmatrix}, \mathbf{x}_2(t) = e^{2t} \begin{pmatrix} 0 \\ 1 \\ 0 \end{pmatrix}, \mathbf{x}_3(t) = e^{3t} \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix}$$

### Пример 2: Кратное собственное значение (диагонализируема)

$$\frac{d\mathbf{x}}{dt} = \begin{pmatrix} 2 & 0 & 0 \\ 0 & 2 & 0 \\ 0 & 0 & 3 \end{pmatrix} \mathbf{x}$$

Найти ФСР.

**Решение:**

Собственные значения: $\lambda_1 = 2$ (кратность 2), $\lambda_2 = 3$ (кратность 1)

Для $\lambda_1 = 2$: два линейно независимых собственных вектора:
$$\mathbf{v}_1 = \begin{pmatrix} 1 \\ 0 \\ 0 \end{pmatrix}, \mathbf{v}_2 = \begin{pmatrix} 0 \\ 1 \\ 0 \end{pmatrix}$$

Для $\lambda_2 = 3$: один собственный вектор:
$$\mathbf{v}_3 = \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix}$$

ФСР:
$$\mathbf{x}_1(t) = e^{2t} \begin{pmatrix} 1 \\ 0 \\ 0 \end{pmatrix}, \mathbf{x}_2(t) = e^{2t} \begin{pmatrix} 0 \\ 1 \\ 0 \end{pmatrix}, \mathbf{x}_3(t) = e^{3t} \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix}$$

---

## ❓ Интерактивные вопросы

**Q1:** Когда используются обобщенные собственные векторы?
<details>
<summary>Ответ</summary>
Когда матрица не диагонализируема, т.е. нет достаточно линейно независимых собственных векторов для кратных собственных значений.
</details>
